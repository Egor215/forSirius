# Инструкция по установке и запуску бота для сжатия текста

## Установка зависимостей

Перед запуском бота необходимо установить все требуемые библиотеки. Откройте командную строку в папке с проектом и выполните: 

pip install -r requirements.txt


Проект использует следующие библиотеки:
- aiogram - для работы с Telegram Bot API
- python-docx - для обработки Word документов
- python-dotenv - для работы с переменными окружения

## Получение токена Telegram бота

1. Найдите в Telegram бота @BotFather
2. Отправьте ему команду /newbot
3. Следуйте инструкциям для создания нового бота
4. После успешного создания бота, вы получите сообщение с токеном (строка вида "123456789:ABCdefGHIjklmNOPQrstUVwxyz")

## Настройка и запуск бота

1. Создайте файл `.env` в папке проекта и добавьте в него полученный токен: BOT_TOKEN=ваш_токен


2. Убедитесь, что у вас установлен Python 3.12 (рекомендуемая версия)

3. Запустите файл `compress.py` с помощью Python

4. Перейдите по ссылке на вашего бота (указана в сообщении от @BotFather) и отправьте команду /start

5. Бот готов к работе!
